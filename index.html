CAMPING GUIJO DE ÁVILA

﻿

El Camping Guijo de Ávila es un centro de actividades para adultos y para niños. Puedes hacer muchos cosas con las instalaciones que voy a citar ahora:

  • Pistas multiusos baloncesto / fútbol
  • Pared de escalada con rocódromo
  • Café Bar con baños
  • Dos centros multiusos para comer, jugar...
  • Baños / cambiadores
  • Una rampa con acceso al río para la actividades acuáticas
  • Cabañas de alojamiento muy básicas para quedarte allí a dormir
  • Apartado adaptado para hacer tiro con arco
  • Canoas
  • Objetos perdidos / Enfermería

En cada actividad hay un profesor experto para ayudarte a aprender y ha hacer todas las actividades y en caso de lluvia te llevarían a uno de los centros multiusos a hacer otra actividad.
